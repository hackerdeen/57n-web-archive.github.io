Howdy Howdy party people!

I've schedulued in an Open Street Map night tonight, just to get there
relevant interested folks together to look at/talk about/play with OSM
stuff again!

If you know what OSM is and want to come and play, awesome - see you
tonight at 7ish!

If you don't, here's a description of the project I wrote previously:

OpenStreetMap (OSM) is a user editable map of the world, released with
an open content license. In the same way that wikipedia is a free, user
generated collection of all human knowledge, OSM strives to be a
free, user generated collection of all human environments.

If you've not seen it, have a browse of Aberdeen! 
http://www.openstreetmap.org/#map=13/57.1433/-2.0963

If you'd like to learn more, swing by and have a chat about anything -
contributing to the map, using the data in novel applications or discuss
how open licensing works in a map situation!

I'll be looking at updating a bit of the Westhill area, and talking
cheerily to all comers, as ever!
