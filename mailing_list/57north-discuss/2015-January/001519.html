This is what hackr is all about. The space's Etherpad instance is run by
Hibby on hackr and anyone that would like to set up a service is welcome to
do so. We also have SNI enabled, so anyone that would like to pay for a
certificate for their service is welcome to do so. Ed has also volunteered
to provide the funds for certificates for our front facing website and wiki,
and so those will use paid certificates.

it. I think it's important that this system is kept with access by directors
only.

If you would like an account on hackr, ping me an email and I'll create one
for you.
