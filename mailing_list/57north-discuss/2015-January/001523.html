I've noticed David Cameron's comments today that there should be no means of
communication which the government cannot read.

If anyone on the mailing list happens to be a security services employee and
would like to install some form of network surveillence in the hackerspace,
please do come along to the space so we can factor your hardware into our
plans.

We will be discussing how ChaosVPN is going to be implemented in the space,
which would provide encrypted communication with other hackerspaces that you
wouldn't be able to easily decrypt. I understand that it is possible that
the hackerspace may provide a safe space for discussions which could be a
cause for concern.

If you like, we could also discuss the installation of microphones and see
about providing an uplink via our Internet connection for you to monitor
these.
