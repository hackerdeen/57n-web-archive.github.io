pffffft.. I can make a quick prototype this evening/Tues and come up with
costings. Wasn't planning it becoming a 'MakeIt-Move' workshop, although it
can be a test for it.  I only have 12 vibro motors on order but for
testing, the rest can be pound-shop stuff and thermoplastic.  No uC is
really needed as it's just a simple Braitenberg vehicle.
<http://en.wikipedia.org/wiki/Braitenberg_vehicle>

Would lile to test somethings out tho -

1) I was thinking about throwing in an LDR before the motor as part of a
voltage divider. This will allow people to control the speed of a single
bug-bot via a £1 laser pointer (kinda nice) but moreover, allows for remote
steering when two robots are tied together - that might make the battles
more skillfull!

2) Using the conductive paint on ground feelers and areas on a track, we
can have very simple line-following behaviours. an uC can be handy here,
but you can still do it with just passive components.

Anyone have any old mobile phone vibro-motors that we can test with this
evening?
