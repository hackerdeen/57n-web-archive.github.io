Hi

I don't mind doing a Tuesday up talk, some random ideas I could do are...

   - Web development state of the nation - and insider's view on web
   development generally and locally.  I did this at TechMeetup a couple of
   months ago, so one or two people have maybe seen this already.
   - Inside a healthy Open Source project and community - In October I went
   to Joomla Day (Joomla is an Open Source CMS, Joomla Day is the annual
   conference), so this would be a talk about various Open Source project
   governance models and practical experiences of the community.
   - An introduction to the Joomla CMS.  This would be a quick fly though
   the Joomla CMS system and platform.  So we'd talk though use cases, user
   experiences, installation, configuration, code structure, developing
   extensions, coding up templates, tool-sets, skill-sets and comparisons to
   other CMS systems.

...so if anyone would be interested in one or more of them, let me know.
Don't expect anything too cool or TED-like though, just blurb about the
things I'm into :)
