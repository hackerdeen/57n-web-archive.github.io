Hi, Iain.

Server PSUs are overspec'ed for at least two reasons: One is peak 
start-up 'inrush' currents, and another is a safety margin for 
reliability because if you run a PSU at close to its designed maximum 
capacity you will shorten its life in comparison to running an 
over-spec. PSU with the same load. This is a 'cheap' alternative to 
using redundant PSUs.

I've got a 1,000W PSU in my main Minke server "sei", which generally 
pulls less than 500W total (incl. disks and GPGPU) when it's working hard...
