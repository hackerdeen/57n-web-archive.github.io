Hi,

Today I would like to introduce hackr, a project I've been working on for
members of the space.

hackr is a quad-core (8 threads) FreeBSD machine with 32GB of RAM with
accounts available, subject to approval, to members of 57North Hacklab. It
is hosted in an OVH datacenter in France with a high-speed Internet
connection and currently offers the following services:

 * shell access
 * static website hosting
 * email
 * running background processes (e.g. an IRC bouncer)

To request an account on hackr, send an email to irl at fsfe.org. You may
include a GPG fingerprint if you wish your password to be emailed to you
encrypted.

This project isn't just about providing a service to members and leaving
them on their own, but also providing guides on how to do things in a Linux
environment to allow people an entry point to using Linux. To that end,
there is a wiki page: https://57north.co/wiki/Hackr which lists a series of
guides. I hope to see that list grow in the coming weeks.

If you would like new software to be installed, please send your request to
the 57north-discuss mailing list. As long as your request isn't ridiculous,
it's likely to be installed. It's even more likely to be installed if you
can tell me where to find it in the FreeBSD ports tree, but this isn't
required.

Iain.
