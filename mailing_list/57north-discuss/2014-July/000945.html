Hi,

We now have a MediaGoblin instance. Media isn't just pictures. MediaGoblin
also supports audio files, videos, books and presentations or even 3D-models
and ACSII-art. On the backend, it can use powerful tools to transcode or
convert your files. You can find out more about MediaGoblin here:

http://mediagoblin.org/pages/tour.html

This has been set up in response to Flickr being dreadful for hosting photos
but this is definitely not a finished solution. Do experiment with it, but
things might break until it's set up properly.

There may not be anything in the web interface for uploading multiple images
at once but there is an API that can be used to do that, you just have to
write a bit of code. I'm sure either I, or someone else, will publish some
code for doing that soon.

Carl - are you still looking at an LDAP server? It would be really useful to
have accounts using LDAP for this so people don't need to sign up for
another account.

You can find the instance at:

  http://media.hackr.org.uk/

Feedback is welcome.

Iain.
