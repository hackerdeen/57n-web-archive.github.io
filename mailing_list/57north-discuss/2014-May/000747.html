The MRCC use a geographical based direction finder system on HF, If my current trip hasn't driven me mad beyond reason. It compares the signal level at different antennas spaced out along the coast, and from that they use a similar principle to a Vein diagram to work out where the signal is coming from. It would require a minimum of 3 stations with known coordinates in a given grid, but it can be done without any relative boundaries using this method.
 
Hope this helps
