Hi all.

Last night was a relatively quiet open night so Peter and I worked on
installing a 12v power distribution solution to replace the 3 separate
supplies that formally powered the lower-level RGBW strip-lighting.
It's secure, vented, safety-labelled and tested fine but is
disconnected for now until I can double check if active cooling is
required. Details are up on the wiki [1] and the call for
comments/objections is open.

People have been talking about adding a space 12v network for a while
and I don't claim that this is a total solution. It is however, handy
in the medium term for getting all the current RGBW lights back up as
well as powering possible wall mounted MQTT control panels I am
planning [2]. If the current setup raises no objections, we'll leave
it in place, probably add more cooling and repeat the setup same for
the upper floor at some point. Otherwise it can all easily be removed.

Cheers,
Nordin
