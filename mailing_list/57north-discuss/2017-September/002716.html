Hi All,

I'd like to get some idea of how much interest there would be in shitty
virtual machines for hackerspace members.

Each virtual machine would have 1 virtual core, 96MB RAM and 20GB disk
space. These would be limited to one per member and be first come first
served.

You would have a single IPv6 address with no firewall, but no IPv4
connectivity at all. A subdomain of hackr.org.uk will be created for
your host, but feel free to set up your own domain to point to it if you
like.

If there is interest and a trial proves that these virtual machines have
value, then I would look to accept donations with an aim of improving
the infrastructure. The shitty virtual machines will be free for
hackerspace members independent of donations as long as resources are
available.

Thanks,
Iain.
