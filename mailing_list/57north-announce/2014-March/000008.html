Sup,

I will be at the space this Saturday from 2pm. All the infrastructure
work is over and the space is now business-as-usual again. If you plan
to come in, send a post to the discussion list with what you plan to be
working on.

I will be working on the sensors management module of the space
management system using node.js and express. Express is actually pretty
nice for this, so I hope to get it to a point where it can be deployed
tomorrow.

There is just one current space issue, the Ethernet is currently not
connected. There is one Ethernet port available if you want to use it to
connect to the Internet, but we haven't managed to patch the phone line
into the rack yet.

If there is a need for more Ethernet ports, we can always run a switch
off the router temporarily.
