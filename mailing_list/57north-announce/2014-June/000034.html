Hi,

The -announce and -discuss list digests are now being sent using MIME format
instead of a concatenated plaintext file. This should allow (useful) mail
clients to allow you to reply to individual messages keeping threads intact
and having messages directed to the appropriate reply address.

Useful mail clients include: mutt (and possibly Thunderbird) but do not
include GMail's web interface, Yahoo! mail or Facebook mail (seriously, why
is that even a thing?).

Please let me know if this causes any problems, and if they can't be fixed,
we can change this back.

Iain.
